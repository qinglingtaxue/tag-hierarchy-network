# 可视化规范详细文档

## 背景与线条

| 属性 | 值 |
|------|-----|
| 背景渐变 | 径向渐变 `#1a2540` → `#0f1525` → `#080a12` |
| 中心光环 | 半径 180px，`rgba(140, 170, 220, 0.15)` |
| 中心光点 | 半径 60px，`rgba(180, 200, 240, 0.4)` 渐变 |
| 背景星星 | 200 个随机分布的小圆点，透明度 `0.2-0.7` |
| 默认线色 | `rgba(120, 130, 160, 0.15)` |
| hover 族谱线 | `rgba(255, 200, 100, 0.8)`，线宽 `2` |
| hover 非相关边 | `rgba(100, 110, 140, 0.05)`，线宽 `0.3` |

## 节点大小（按频次）

```javascript
size = 3 + (freq / maxFreq) * 10  // 范围 3-13（更精致）
```

## 节点颜色（按层级，柔和色调）

| 层级 | 颜色 | HSL |
|------|------|-----|
| Level 0（核心） | 柔金 | `hsl(45, 70%, 58%)` |
| Level 1（重要） | 浅蓝 | `hsl(200, 45%, 70%)` |
| Level 2（普通） | 浅绿 | `hsl(140, 40%, 60%)` |
| Level 3（次要） | 灰蓝 | `hsl(210, 35%, 75%)` |
| Level 4（边缘） | 浅灰蓝 | `hsl(220, 25%, 80%)` |

## 布局（弧形彗星尾）

| 层级 | 半径范围 |
|------|----------|
| Level 0 | 中心 `20 - 60px` |
| Level 1 | `80 - 130px` |
| Level 2 | `150 - 210px` |
| Level 3 | `230 - 300px` |
| Level 4 | 外围 `320 - 400px` |

### 弧形布局算法

1. 每个 L0 根节点分配一个主方向角度
2. 同一根节点下的子孙节点沿弧线向外扩散
3. 扩散角度随层级增加：`spreadAngle = 0.15 + level × 0.12`
4. 形成"彗星尾"效果，每个分支独立延展

### 约束

- 层级越高（L0）越靠近中心
- 力导向防止重叠
- 节点可单独拖拽（限制在画布边界内）
- 边界限制：30px 边距

## 交互

- ✅ 滚轮缩放：`0.3 ~ 4`
- ✅ 按住空白拖拽平移
- ✅ 按住节点单独拖拽（边界限制，不超出画布）
- ✅ 按钮：放大(+0.3)、缩小(-0.3)、重置视图
- ✅ hover 高亮完整族谱（祖先→当前→后代→后代的后代...）
- ✅ 节点边界限制：30px 边距，防止超出网页

## 层级连接规则

**严格相邻层级连接**：
- Level 0 → Level 1
- Level 1 → Level 2
- Level 2 → Level 3
- Level 3 → Level 4

禁止跨层级连接（如 L0 直接连 L3）。
