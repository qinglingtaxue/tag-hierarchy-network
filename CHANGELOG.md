# Changelog

本文档记录 `tag-hierarchy-network` 的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.2] - 2026-03-27

### 新增
- 5 种通用场景调用指南（组织架构、产品分类、知识图谱、技能树、标签体系）
- 自定义配置 `window.HIERARCHY_CONFIG` 支持
- 完整的验收清单

### 优化
- 弧形彗星尾布局算法优化
- 核心发现面板 UI 改进
- 文档结构重组，更易阅读

## [1.0.1] - 2026-03-26

### 修复
- 节点拖拽边界溢出问题（添加 30px 边距限制）
- hover 族谱高亮在深层节点时的性能问题

### 优化
- 柔和层级配色微调（柔金 → 浅蓝 → 浅绿 → 灰蓝 → 浅灰蓝）

## [1.0.0] - 2026-03-25

### 新增
- 初始发布
- 径向层级网络图核心功能
- 弧形布局（彗星尾效果）
- 5 级层级支持（L0-L4）
- 交互功能：缩放、拖拽、族谱高亮
- 核心发现面板（Top 3 分支汇总）
- 数据格式：nodes + edges JSON
- 三种数据加载方式：URL参数、全局变量、同目录文件
- MIT 许可证
